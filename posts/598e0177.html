从零开始安装Arch Linux

前置配置

将下载的arch linux ISO文件烧录到U盘中

  • 如果在linux环境中，可以直接使用dd命令
  • 如果是在windows环境中，可以使用usbwriter, rufus, etcher 等工具

1. 配置网络

  • 在无线环境中，可使用以下命令配置网卡连接wifi，若是在有线的情况下则可忽略下面的步骤
  • 更新系统时钟

2. 磁盘操作

2.1. 磁盘分区

  • 对于UEFI/GPT
mount point partition partition type
/mnt/boot /dev/esp_partition efi
/mnt /dev/root_partition linux
/mnt/home /dev/home_partition linux
[swap] /dev/swap_partition linux swap
  • 对于BIOS/MBR
mount point partition partition type
/mnt /dev/root_partition linux
/mnt/home /dev/home_partition linux
[swap] /dev/swap_partition linux swap

2.2. 格式化分区

2.3. 挂载分区

3. 配置系统

3.1 进入系统前

  • 将mirrorlist中的镜像服务器地址改为中国，加速后续软件下载速度
  • 安装一些重要的软件包
  • 生成fstab文件

3.2 进入系统

  • 更改root目录

如果在前面的硬盘分区中你不想创建一个单独的swap分区，你可以在这一步创建一个swapfile文件来代替swap分区。

3.3 进入系统后

  • 设置时区
  • 设置硬件时钟同步
  • 本地化设置
  • 设置主机名
  • 设置本机域名解析

configure boot loader

grub

  • for BIOS/MBR
  • for UEFI/GPT

systemd-boot

install some tools

启动系统服务

  • 设置root的用户密码

千万别忘记了这一步，否则之后重启进入不了系统。当然你也可以借助U盘进入到系统中，重新挂载磁盘，然后设置密码，但是很麻烦。这可是小编血的教训。

重启

  • 首先使用exit命令退出当前环境
  • 使用umount -R /mnt卸除之前挂载的目录
  • 最后，使用reboot命令重启

系统安装后的配置

关于用户的设置

  1. 新增用户并添加到wheel用户组
  1. 为新增用户添加密码
  1. 给wheel用户组中的用户添加sudo权限

如果不安装vi，安装其他基于vi的编辑器，如vi，neovim等，可以考虑将其软链接到vi，以便于visudo命令的使用。
